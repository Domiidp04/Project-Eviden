Para la base de datos, utilizo una base de datos postgres alojada en un docker. Para que se conecte el back con la base de datos, lo unico que tienes que hacer es crear la base de datos llamada testdb, no hace falta los create, se crean automaticos.
Lo que si te quiero pasar, es el insert que lo que hace es convertir el usuario que has creado en usuario administrador(InsertUserAdmin.txt). Mi recomendacion es que registres un usuario admin, y dos usuarios user(Se crean por defecto cuando registras un usuario), para que tengas facilidad a la hora de comprobar las vistas.
